for data analysis and reporting.

- Filtered, grouped, and aggregated data
- Used JOINs to combine multiple tables
- Calculated basic KPIs for business insights

- SELECT, WHERE, ORDER BY
- GROUP BY, COUNT, CASE
- INNER JOIN, LEFT JOIN
- Basic KPI calculations

## Project Structure

- queries/

  - joins.sql – JOINs between tables
  - group_by_kpis.sql – KPI calculations
- data/ – example data (optional)
