# SQL Data Analysis

Dieses Repository zeigt meine grundlegenden SQL-Kenntnisse
im Bereich Datenanalyse.

## Inhalte

- Schreiben von SQL-Abfragen
- Filtern und Sortieren von Daten
- Aggregatfunktionen (COUNT, GROUP BY)
- Strukturierte Analyse von Daten

## Projektstruktur

- queries/ – SQL-Abfragen
- data/ – Beispieldaten

## Ziel

Teil meiner Umschulung im IT-Bereich
mit Fokus auf Data Analysis.
